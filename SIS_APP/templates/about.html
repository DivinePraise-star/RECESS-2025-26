About SIS

Student Information System

A lightweight Flask application for registering, viewing, editing, and managing student records in one place.

Built with Flask + SQLite

What it does

The Student Information System (SIS) is a web-based application built using the Flask framework. It provides a platform for managing student registration and information retrieval.

The interface is designed to keep the core student workflow simple: register a new student, review their profile, edit details when needed, and remove records when appropriate.

Key Features

  • Home Page: Landing page with navigation links.
  • About Page: Information about the application.
  • Registration Form: Collects student details using POST method.
  • Dynamic Routes: URL parameters for personalized pages.
  • Database Storage: SQLite for persistent data storage.

Technologies Used

Backend Python with Flask framework
Frontend HTML5, CSS3
Templating Jinja2
Database SQLite with SQLAlchemy ORM
